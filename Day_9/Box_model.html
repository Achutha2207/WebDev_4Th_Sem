Box Model

For any element there is

  • Margin
  • Border
  • Padding

Margin

It is the outter most layer in a box

Border

Border is an attribute to add boundary to the elements

Padding

Padding is the inner most layer of an element

Normal Tags {0,0}
Boxed Tags {10,10}
box-sizing:border-box; Is a declaration used to say to the Compiler
That the Height and Width attributes of the respective box is the sum of
  • Contents
  • Border
  • Padding
Elements [But margin is not included because it is outside the box]
Essentially saying to the compiler that this is the Total Size Of The Element/Tag
But if we dont include the box-sizing then the width and height are given only to the contents
And not the padding / border / margin They are given seperately

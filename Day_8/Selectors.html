Element Selectors

It is used to style all the elements universally eg:

div{color:blue;}

This selection makes all the tags named div to be
styled in a specific manner specified inside the CSS function



*****************************************************************************************

Class Selectors

When we mention s class attribute inside a tag then
We can accsess those tags using the . operator and styling
them using css for example

<p class="green"> </p>
We can access the above p tag using .green in CSS file
eg: .green{color:aqua;}

*****************************************************************************************




Id Selectors

We can give id to a specific tag and access that tag using
# operator in CSS file and style it further for example

<span id="guts"> </span>
I can use # operator to access id's of any tags eg:

#guts{color:grey;}
*****************************************************************************************


Child Selectors


If I want to say that I only want to style the children of X Tags
Then I use chile selectors which will only style the specified
child of a specified parent eg:

span > div { color: wheat;} 
This will only style the div tag only when its a parent of span
tag and not a child of any other tags eg:

<span>
<nav>
<div>
This div cant be considered styled because its not the child of span
but its a descendent of span tag {Inside a nav tag}.

*****************************************************************************************






Descendent Selectors


Used to style if I need to style the descendents of a specific tag we
simply remove the > [Child of symbol] symbol in the Child selector
It has syntax as :

parent child {color:blue;}

*****************************************************************************************








Universal Selector


We use it to include all the elements in the HTML file and assign a specific
functionality to it for eg: allign all elements to left.
All elements of HTML file can be accessed using * operator eg:

*{
margin:0;
padding:0;}
*****************************************************************************************






Pseudo Selectors


Selectors which are fake : Where styling of the elements are not fixed but
They change as the client interacts with the website for eg: I want the
Hyperlink to be red after it is visted by the client / I want to add a background
color to the url whenever it is hovered {by a mouse}.

Styling of the elements is dependent on specific actions performed by the client
